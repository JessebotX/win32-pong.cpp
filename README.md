# Win32 Pong

(WIP) Modern C++ implementation of Pong on Windows. No dependencies except for the Win32 API.

# Building

You can run the provided batch file (requires CMake, Ninja, and a C++ 23+ compiler (GCC, clang, cl, etc.)) to quickly produce a working executable.

In cmd.exe:

```
build.bat
out\w32-pong.exe
```
